# Parcoursup Challenge
In this challenge, we will be dealing with a Parcoursup database. 
Parcoursup is the national algorithm that matches high school students with universities and schools in order to do the best possible allocation of orientations. In 2020, it manages nearly 660,000 students and 15,500 orientations. 

The goal of this challenge is to predict the pourcentage of admitted female among all the admitted students depending on some characteristics of the school, track, city, etc. It is a regression problem.
